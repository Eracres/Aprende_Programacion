# 🧪 Ejemplo 6: Iframe para mapa de Google Maps

## 🎯 Objetivo
Aprender a insertar mapas interactivos usando `<iframe>`.

## 📁 Ruta: /src/Ejemplos/Enunciados/Ejemplo_6.html

```html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Mapa de ubicación</title>
</head>
<body>
  <h2>Nuestra ubicación</h2>
  <iframe src="https://www.google.com/maps/embed?pb=!1m18..."
          width="600" height="450" style="border:0;"
          allowfullscreen="" loading="lazy"
          referrerpolicy="no-referrer-when-downgrade">
  </iframe>
</body>
</html>
```

---

## ✅ ¿Qué hace este ejemplo?
Permite mostrar una localización geográfica con interactividad, sin salir del sitio web.

---

## 🧠 Conceptos aplicados
- Integración de servicios externos
- Seguridad con `referrerpolicy`
- Interfaz enriquecida con `iframe`
